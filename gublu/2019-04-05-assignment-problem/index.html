    explain like i'm five - Assignment Problems

    4 minutes read

    As an HR manager, you’ve for vacant posts at your company - say (Post A, Post B, Post C, Post D). After long advertisement, only 4 peoples (Person 1, Person 2, Person 3, Person 4) turned up for interview. Each have them have their salary expectation for each posts as given below :

    person 1 : 8L, 6L, 5L & 7L for Post A, B, C, D respectively. person 2 : 6L, 5L, 3L & 4L for Post A, B, C, D respectively. person 3 : 7L, 8L, 4L & 6L for Post A, B, C, D respectively. person 4 : 6L, 7L, 5L & 6L for Post A, B, C, D respectively.

    Your task as HR Recruitment Specialist is to find right candadates for each post such that total cost to company is minimized. (let’s assume all are well qualified for their jobs and focus on math here)

    Above problem can be represented in simple matrix format -

      Post A Post B Post C Post D
    1 8 6 5 7
    2 6 5 3 4
    3 7 8 4 6
    4 6 7 5 6

    This is a typical assignment problem. One person gets assigned to one job. Note that -

    1. Once a person, A has been assigned a job 1, he (A) can’t be assigned to another job. And job (1) can’t be assigned to another person.
    2. if a person reduces / increases their salary expectation by a contant for each job, it’d not affect the what job he’d get. i.e. you can subtract a constant value from each row and answer will not change.
    3. similarly, if all person has low salary expectation for a job, it’d not affect what they will get (job assignment will remain unchanged) i.e. you can subtract a constant from each column and answer will not change.
    4. from (2) and (3) you can observe, subtracting a constant value from entire table will also not affect the solution.

    Based on above mentioned facts, we’ll try to solve the problem - We want atleast one zero in each column and one zero in each row. We use (2), (3) and (4) to achive that. step 1: we subtract the row-minimum from each row. (You can start with subtracting a constant from entire table as well)

      Post A Post B Post C Post D subtract from row
    1 8 6 5 7 5
    2 6 5 3 4 3
    3 7 8 4 6 4
    4 6 7 5 6 5

    step 2: All rows have one zero. But column A, B, D doesn’t have zeros. Let’s subtract column minimum from each column -

      Post A Post B Post C Post D
    1 3 1 0 2
    2 3 2 0 1
    3 3 4 0 2
    4 1 2 0 1
    subtract from column 1 1 0 1

    step 3:

      Post A Post B Post C Post D
    1 2 0 0 1
    2 2 1 0 0
    3 2 3 0 1
    4 0 1 0 0

    Now we have one zero in each column and row. Remember that our solution (which person gets which job) is same as we’d have got from first matrix. Clealy, in step 3 matrix, we can find a assignment such that there’s zero apparent cost to company. Actual cost to company has to be calculated from initial matrix.

    Number of zeros in row denotes possible jobs that person can get - 1 can get B, C. 2 can get C, D. 3 can get only C. 4 can get A ,C, D. We allocate job first to that person who has least number of options. in this case 3.

    One solution can be -

    • person 3: Job C. Now Person 3 can’t be assigned any more jobs ( we strike off row 3 ) and Post C can’t be assigned to any other person ( we strike off column for Post C )
    • Looking at remaining matrix, person 1 has only one option left, we can assign Person 1: Post B. Strike off row 1 and Column B as Post B is filled and person 1 is occupied.
    • Now Person 2 get Post D. strike off row 2 and column D.
    • Person 4 gets Post A.

    Assignment is : 1 gets B, 2 gets D, 3 gets C, 4 get A. total cost is : 6+ 4+ 4 + 6 = 20 (look at 1B, 2D, 2C & 4A in table in question) —- In case such assignments are not possible (suppose, in case of person 4, there are no remaining zeros in row), we say no solution exits to the assignment problem.

    If number of Jobs is more than people OR number of peoples are more than job, It is called an Unbalanced Problem. In such cases we fake jobs (columns) or peoples (row) with zero entries. And then solve the problem as before.
